Bar Chart using Randomly Generated Dataset

This bar chart will look different each time the page is refreshed, because it uses a function that generates a dataset of 25 random numbers with values 1-30.

Once we have our random dataset, D3.js overrides the `height` property of our div.bar css style. So if an individual data point is '25,' it will make that bar 25px high (actually, 25px is tiny...so we made it data * 5 for readability).
